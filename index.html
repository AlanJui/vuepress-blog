中道部落格

使用 VuePress 打造的部落格

起始指南 →

操作簡便組織容易

藉由 VuePress 的特性，使系統技術文件的撰寫、編排與瀏覽，操作方法變得簡單，但資訊之組織更具結構化。

沒有制約擴展無限

VuePress 底層技術為： Vue + Webpack，功能不足之處，使用者可透過 Vue Components in Markdown 自行擴展.

佈署簡單效能快捷

經由 VuePress 製成之文件，檔案格式為「靜態網頁」。放入 HTTP 伺服器變可操作使用。網頁呈現採用 SPA （Single Page Application），可加快顯示速度，便於瀏覽操作。

# 文章編排體裁

部落格內文章，依下列體裁向讀者標明閱讀時應留意之重點：

操作秘訣

類似英文 Tip 之意，當操作之執行有多種方法時，除了文章示範的方法外，第二種執行方法，說明於「操作秘訣」內。

小心留意

類似英文 Warning 之意，提醒讀者需要小心留意之處，因為該操作若執行錯誤，將導致系統運作發生異常。

危險警告

類似英文 Dangerous Warning 之意，表示該操作執行前須再三複核，待確認沒問題後才能進行。因為執行後的結果，沒有任何挽回的餘地。
